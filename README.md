# Weather Dashboard

This repository contains a Flutter weather dashboard project in the
`weather_dashboard_app_full` directory.

## Project

- App: `weather_dashboard_app_full`
- Tech: Flutter (Dart)
- Includes: city search, current weather card, 7-day forecast, and weather icons

## Getting Started

1. Ensure Flutter SDK is installed and available in your PATH.
2. Open a terminal in `weather_dashboard_app_full`.
3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
